# Installation

> git clone https://github.com/myleaningstuff/safeword
> cd safeword
> yarn
> truffle deploy

Find out the uniswap router address in your network.

> truffle console
> safeword = await SafeWord.deployed();
> safeword.setUniswapRouter(<uniswap_router_address>);
